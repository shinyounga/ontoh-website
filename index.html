How it works

ONTOH가 현장을 지키는 방법.

자율 순찰 모드 활성
01

AI가 스스로 카메라를 제어합니다

PTZ 카메라가 자동으로 위험 구역을 순찰하며 사각지대를 실시간으로 해소합니다. 수동 조작 없이도 현장 전체를 커버합니다.

24시간 무인 자율 순찰
위험 감지 시 자동 줌인/포커스
기존 CCTV 장비와 호환 가능
02

위험을 감지하고, 맥락을 판단합니다

단순 객체 인식을 넘어 RAG-LMM 기반의 복합 상황 판단으로 오경보를 줄이고, 실제 위험 상황만 정확하게 알립니다.

20,000+ 자체 학습 데이터 기반
안전 장비, 중장비 등 다중 위험 감지
산업안전보건법, 건설 안전 법령 및 규칙 자동 매칭
위험 상황 감지
보건 리포트
개인 맞춤형
03

근로자 건강까지 데이터로 관리합니다

건강검진 결과를 자동 연동하여 고위험군을 즉시 분류합니다. 현장별, 개인별 맞춤 보건 리포트로 질병형 산업재해까지 사전에 대응합니다.

건강검진 데이터 자동 연동
질환 민감군 자동 분류
현장 맞춤형 보건 리포트 생성
