Core Feature

AI 기반 CCTV 자동 제어

위험 상황이 감지되면 PTZ 카메라가 자동으로 해당 지점을 추적합니다.
관제 요원이 수동으로 카메라를 조작할 필요 없이, AI가 최적의 화각을 계산하여 카메라를 이동시키고 줌인합니다.
사각지대 없는 능동형 관제 체계를 구현합니다.

자동 추적
자동 줌인
360도 커버

RAG-LLM 복합 상황 판단

단순 패턴 매칭이 아닌 RAG 기반 대형 언어 모델이
영상 속 상황을 종합적으로 판단합니다.
여러 위험 요소가 동시에 존재하는 복합 상황에서도 정확한 분석과 우선순위 판단이 가능합니다.

현장 맞춤형 안전 교육 자료 생성

실제 현장에서 발생한 위험 사례를 기반으로 교육 자료를 자동 생성합니다.
추상적인 교육이 아닌, 작업자가 매일 마주하는 현장의 실제 상황으로 안전 의식을 높입니다.

법령 / 규정 / 규칙 준수

산업안전보건기준에관한규칙 제43조(추락 방지), 제32조(보호구 착용) 등
관련 법령을 AI가 자동으로 매칭합니다.
